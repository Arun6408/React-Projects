### Basic Brand Page

A simple brand page to showcase your brand's identity.

#### Setup Instructions
1. Navigate to the project directory:
    ```sh
    cd basic-brand-page
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Start the development server:
    ```sh
    npm start
    ```
